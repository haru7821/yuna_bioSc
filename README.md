# 🧬 고2 생명과학Ⅰ 모의 퀴즈 및 30일 완성 챌린지

고등학교 2학년 생명과학Ⅰ 교과 과정의 핵심인 **신경계 & 근수축**, **항상성 유지 & 면역**, **유전** 파트를 효과적으로 학습할 수 있도록 설계된 웹 기반 교육 플랫폼입니다. 교과서 및 최신 출제 기준을 바탕으로 고품질의 문항 풀을 구축하고, 성취도 평가 분석 시스템과 30일 단기 학습 플래너(챌린지)를 제공합니다.

본 웹사이트는 별도의 빌드 단계나 백엔드 서버 없이 브라우저에서 독립적으로 동작하는 **Single Page Application(SPA)**으로 구성되어 있어, 로컬에서 즉시 학습하고 깃허브 페이지(GitHub Pages) 등으로 즉각 배포할 수 있습니다.

---

## 🌟 주요 기능

### 1. 단원별 모의 퀴즈 (Quiz Mode)
- **선택 학습 기능**: 교과과정에 맞춤 설계된 **7개 중단원** 중 원하는 영역만 골라 선택적으로 퀴즈를 출제할 수 있습니다.
- **20문항 무작위 추출**: 선택한 중단원들의 문제 풀에서 중복 없이 무작위로 20개의 문항을 출제하여 매번 새로운 문제를 푸는 효과를 줍니다.
- **풀이 타이머**: 풀이 소요 시간을 기록하여 시간 안배 연습이 가능합니다.
- **상세한 정오답 해설 피드백**: 보기 선택 시 실시간으로 정오답 상태를 시각화하고 상세 해설 카드가 노출됩니다.

### 2. 수능/내신 등급 환산 및 성취도 진단 (Analytics Dashboard)
- **등급 평가**: 수능/모의고사 원점수 등급 컷 기준을 적용하여 1~9등급 성취도를 자동 판정합니다.
- **분야별 정답률 분석**: 대단원 1의 3개 중단원 및 대단원 2·3의 10개 소단원(총 13개 단원) 각각의 실제 출제 문항 수와 정답률을 보여주어 학생의 취약 부위를 세밀하게 진단합니다. (퀴즈에서 제외된 단원은 '미선택' 표시)
- **맞춤형 피드백**: 획득한 등급과 취약점에 따라 맞춤형 공부 방향성을 제시합니다.
- **오답 노트 복습**: 오답이 발생한 문항만 별도로 모아 정답과 상세 해설을 복습할 수 있게 설계되었습니다.

### 3. 30일 완성 학습 챌린지 (30-Day Challenge)
- **13대 세부 단원별 시험 범위 (대단원 1은 중단원, 대단원 2·3은 소단원 분류)**: 
  - **Ⅰ-❶ 생명과학의 이해 (1~2일차)**: 생물의 특성, 바이러스의 특징, 생명과학의 특성과 구성 단계
  - **Ⅰ-❷ 생명활동과 에너지 (3~5일차)**: 물질대사와 ATP, 소화·순환·호흡·배설계 통합 작용, 대사성 질환
  - **Ⅰ-❸ 생태계와 상호작용 (6~10일차)**: 생태계 구조와 환경, 개체군 밀도와 생존 곡선, 군집의 천이와 상호작용, 에너지 흐름
  - **Ⅱ-❶-01. 신경자극전도와 시냅스전달 (11~12일차)**: 신경 세포막의 분극·탈분극·재분극 막전위 변화와 흥분 전달
  - **Ⅱ-❶-02. 신경계의 구조와 기능 (13~14일차)**: 중추 신경계(뇌/척수)의 기능 조율 및 말초 신경계(체성/자율 신경)
  - **Ⅱ-❶-03. 항상성 유지 (15~16일차)**: 호르몬 분비 피드백 루프와 체온·혈당량·삼투압 항상성 조절
  - **Ⅱ-❷-01. 병원체와 방어작용 (17~18일차)**: 세균·바이러스·프라이온 감염성 질병과 인체 1·2차 방어작용
  - **Ⅱ-❷-02. 항원항체반응과 백신 (19~20일차)**: 백신의 예방 기전과 기억 세포 형성 및 ABO/Rh식 혈액형 판정
  - **Ⅲ-❶-01. 염색체와 유전자 (21~22일차)**: 염색체 및 뉴클레오솜 구조, 대립유전자, 가계도 및 다인자 사람의 유전
  - **Ⅲ-❶-02. 생식세포 형성과 유전 다양성 (23~25일차)**: 세포 주기, 체세포/감수 분열과 유전 다양성 원리, 비분리 돌연변이
  - **Ⅲ-❷-01. 생물의 진화 (26~27일차)**: 다윈의 자연선택설에 의한 생물의 적응과 진화 원리 및 증거
  - **Ⅲ-❷-02. 생물의 분류체계 (28일차)**: 생물 분류 계급(계문강목과속종) 및 현대 3역 6계 분류
  - **Ⅲ-❷-03. 식물과 동물의 분류 (29~30일차)**: 선태·양치·겉씨·속씨식물의 분류 및 다양한 무척추/척삭동물 분류
- **매일 학습 및 미니 퀴즈**: 각 일차마다 단원별 요약 노트를 읽고, 3문항의 일일 미니 테스트를 통과해 학습을 완료합니다.
- **데이터 로컬 저장 (`localStorage`)**: 브라우저의 내부 저장소를 사용하여 진행률이 일차별로 저장되며, 새로고침이나 브라우저 재시작 시에도 학습 진도가 보존됩니다.

## 📂 파일 구조

```text
quiz-test/
│
├── index.html       # 메인 SPA 마크업 (레이아웃 및 CDN 리소스 로드)
├── styles.css       # 네온 다크 모드 기반 글래스모피즘 CSS 스타일시트
├── app.js           # 동적 탭 라우팅, 퀴즈 엔진, 등급 연산 및 저장소 제어 로직
├── questions.js     # 대단원별 60개의 고품질 4지선다형 모의고사 문제 은행
├── challenge.js     # 30일 완성 커리큘럼 데이터 (학습 요약 및 데일리 미니 퀴즈)
└── README.md        # 프로젝트 설명 및 실행 방법 가이드
```

## 🚀 실행 방법

### 1. 로컬에서 실행하기
1. 본 저장소를 다운로드하거나 클론합니다.
2. `index.html` 파일을 더블클릭하거나 웹 브라우저(Chrome, Edge, Safari 등)로 엽니다.
3. 로컬 서버 실행 없이도 모든 기능이 즉각 실행됩니다.

### 2. 깃허브 페이지(GitHub Pages)로 배포하기
본 프로젝트는 완전한 정적 파일(HTML, CSS, JS) 구조이므로, 깃허브에 업로드한 뒤 repository 설정(Settings -> Pages)에서 `main` 브랜치를 소스로 지정하여 5분 만에 전 세계에 라이브 웹사이트로 무료 호스팅할 수 있습니다.

## 🎨 디자인 시스템 & UX 특징
- **Premium Aesthetics**: 블랙 네이비(`--bg-primary`)를 기본 배경으로 설정하고, 고대비 네온 컬러(Cyan, Emerald, Violet)로 액센트를 가미해 고급스럽고 집중도 높은 몰입형 다크 모드 디자인을 구현했습니다.
- **Glassmorphism**: 카드는 투명 보더와 `backdrop-filter: blur(16px)` 속성을 부여해 뒤 배경과 부드럽게 융합되는 고급스러운 유리 재질감을 연출했습니다.
- **Micro-Animations**: 사용자가 클릭 및 호버할 때 카드 스케일 조절, 트랜지션 페이드 효과 및 정오답 체크 애니메이션을 정교하게 적용하여 뛰어난 사용자 인터랙션을 제공합니다.
